A simple Flutter application that allows users to browse popular movies, search for movies, and mark favorites.

## Features
- Browse popular movies 
- Search movies by name
- mark any movies as favorites

## Setup Instructions
1. Clone the repository
```bash
git clone https://github.com/amberabhi/movie_explorer.git
```

2. After cloning the repo run the following command 
```bash
flutter pub get 
```

3. Getting TMDB API KEY 
First browse to the following url 
url - https://www.themoviedb.org 
Then sign in using your gmail after that navigate to settings tab and inside setting tab there is a API section copy that API key.

4. Add your TMDB API key
First make a directory inside the root directory named assets and under that directory add a file named .env 
```bash
TMDB_API_KEY=YOUR_API_KEY(THE ONE YOU COPIED IN STEP 3)
```

5. Run the app using `flutter run`
